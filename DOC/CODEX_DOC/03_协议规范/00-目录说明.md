# 协议规范目录说明

本目录存放跨模块、跨进程、跨阶段复用的稳定协议与数据规范。

适合放入本目录的内容：

- 协议版本、消息类型、字段定义和兼容规则
- DTO / payload / fixture 的结构规范
- Model / Controller / View / Runtime 之间的输入输出边界
- 禁止项、校验规则、可扩展字段和后续阶段必须复用的协议语义

不适合放入本目录的内容：

- 设计取舍、架构背景和迁移说明：放入 `DOC/CODEX_DOC/02_设计说明`
- 任务拆分、执行步骤和命令清单：放入 `DOC/CODEX_DOC/04_研制计划`
- 机测记录、验收入口和阶段结论：放入 `DOC/CODEX_DOC/06_测试文档`
